# CLI Grocery Store

A simple command-line interface grocery store application built with Dart.

## Features

- **View Inventory**: Browse available products with stock levels.
- **Add to Cart**: Select items to purchase by name.
- **Checkout**: Calculate total cost including a 10% tax rate.
- **Discount Codes**: Apply coupons for savings.

## Getting Started

### Prerequisites

- [Dart SDK](https://dart.dev/get-dart) installed on your machine.

### Running the Application

1. Open your terminal.
2. Navigate to the project directory.
3. Run the following command:

```bash
dart main.dart
```

## Usage Guide

1. **View Inventory**: Select option `1` to see what is in stock.
2. **Add Item to Cart**: Select option `2` and type the name of the item (e.g., "Apple").
3. **Checkout**: Select option `3` to finalize your purchase.
   - **Discount Code**: When prompted, you can enter `SAVE10` to receive a $10.00 discount on your order.
4. **Exit**: Select option `4` to close the application.
